Detailed Instructions

Create Block Group

Register an account

To start using Bookie Guard, you need a user account. Click here to register one now. After signing up, login here to continue.

Create Block Groups

Block Groups are logical groups created to manage devices that need similar settings. Let's say you have two bookie shops (Bet9ja and MerryBet). You will need two different settings for the shops, hence two Block Groups.

You'll create two Block Groups (Bet9ja Shop and MerryBet Shop). You can use any name but it's best to use a name that relates to the shops. For each block group, enter the number of devices that will be under the group. Select your desired plan and proceed to make payment.

You will be redirected to a payment page to complete the transaction. Once the payment is completed, you'll be redirected to your dashboard. The block group will be automatically assigned an Activation Code and Unlock Code that will be used for activation and deactivation on the devices respectively.

Set Blocking Rules

Add website(s)

Bookie Guard blocks ALL* websites by default. So in order to allow a website to be used, it needs to be added to the allow list.

To add a website, select the block group from Block Group List. Enter the desired website and submit. Voila! Website has been added to the allowed list.

Allow blocked website(s)

There are two ways to allow a previously blocked website.

  • Add the website again to allowed list.
  • Delete the website from list of blocked websites.

Delete website(s)

To remove a website from allowed list of a blockgroup, simply click on the delete button and the website will be deleted. Updates will reflect on all systems within 30 mins, or you can click on Check for Updates in the installed software to pull in updates.

Activate Blocking

Download and install

After setting up your Block groups online, you need to install the program into the computers you want to them to run in.

Download the latest version from here. Install the program and launch it by typing "Bookie Guard" into your search and clicking on Bookie Guard icon.

Activation

You will be taken to the activation page. Here you'll enter your email and Activation Code for the particular Block Group you want to link the computer to. Then click on activate. The details will be validated, and software activated accordingly.

Set Proxy

On successful activation, proceed to set up your system or browser proxy settings accordingly. Set Proxy as localhost and Port as 8888.

Here are instructions for the popular browsers:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

Congratulations!!! Your system is set up now. Go ahead and test by visiting the blocked and allowed websites.
